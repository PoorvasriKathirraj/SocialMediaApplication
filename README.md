SocialMediaApplication
        SocialMediaApplication is a console-based Java project that simulates basic features of a social networking platform. 
        The system allows users to sign up by providing personal details such as name, age, location, and occupation. 
        Registered users can send and accept friend requests to build their social network. 
        The application also allows users to create posts and share updates with their friends. 
        Users can view their friends list and see posts in a simple news feed.
        The program is implemented using Object-Oriented Programming concepts and Java collections for storing data in memory.
